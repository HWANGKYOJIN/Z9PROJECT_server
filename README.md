## 🏷️ 기획의도

Z9은 광고보다 인플루언서의 경험을 더 신뢰하는 현대인을 위한 공동구매 플랫폼으로, 공동구매의 0부터 9까지 모두 다루겠다는 의미를 이름에 담았다. 2022년 현재 인플루언서가 공동구매(이하 공구)를 개최할 때 제품사 쇼핑몰 혹은 N사의 스토어를 이용한다. 전용 플랫폼이 아니기에 공구가 언제 열릴지, 얼마나 팔리는지,  현황을 알 수 없다. Z9는 단순 URL 공유를 넘어 인플루언서의 개성을 그대로 이어가는 데 목표를 뒀다. 공구 예약 시스템, 다양한 상세페이지, 실시간 판매량 표기 등 여러 기능을 도입한다. 소비자는 더 나은 공동구매 방식을 경험하고, 인플루언서는 깔끔한 공구 진행으로 프로페셔널한 이미지를 얻을 수 있다.
